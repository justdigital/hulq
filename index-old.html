HulQ A simple yet powerful XSLT framework

HulQ smashes your XSLT file into many small files, so you have one XSLT per template tag.

After changes to the files, HulQ rebuilds the template again with minified assets.

Get started easily

  1. sudo npm install hulq -g #It needs to be globally available
  2. cd your_project #Go into your created folder
  3. hulq begin your_template.xslt #Create a new hulq project based on your_template.xslt file (you can omit the extension)
  4. #Now it is your turn: You can edit the xslt files (workspace/**/*.xslt)
  5. hulq recover #Create the brand new xslt file with everything together.
  6. hulq deploy #Deploy your recovered xslt to the GSA control panel
